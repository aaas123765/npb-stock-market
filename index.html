ルール説明

まずはこの企画の基本ルールを確認してください。

  • プロ野球選手を株のように売買して資産を増やしていく企画です。
  • 選手の活躍によって価値が変動し、保有選手や売買タイミングで順位が変わります。
  • どの選手を買うか、いつ売るかという判断が非常に重要です。
  • 最終的に資産の大きい参加者ほど上位入賞に近づきます。

ルールをもっとわかりやすく知りたい方へ

詳しいルール解説はYouTubeでわかりやすく説明しています。 初参加の方は、まず動画を見てから参加すると理解しやすくおすすめです。

EVENT 01

先着ボーナス ＋50pt

先着参加者向けの特典です。 早めに参加してくれた方には、スタート時点で 50pt を付与します。 早く始めるほど得をしやすく、初動から有利に動けるイベントです。

EVENT 02

開場記念ボーナス ＋50pt

開場記念として参加者に 50pt を付与するイベントです。 企画スタート直後の参加ハードルを下げ、より多くの方が楽しめるように設定しています。 参加しやすさを高める記念特典です。

EVENT 03

ひな祭り 売却時 1.05倍

ひな祭り記念の期間限定イベントです。 対象期間中の売却時は 1.05倍 で反映されます。 通常よりも利益を伸ばしやすく、短期勝負の面白さも高まる特別イベントです。

EVENT 04

YouTube登録ボーナス ＋25pt

みるスタチャンネルを登録してくれた方向けのボーナスです。 チャンネル登録によって企画の認知が広がり、今後の賞金増額にもつながる可能性があります。 企画を応援しながら、自分にも特典があるイベントです。

EVENT 05

Xリツイートボーナス ＋25pt

Xで対象投稿をリツイートしてくれた方向けのボーナスです。 企画の拡散に協力してくれた方へ還元するイベントです。 参加者が増えることで企画全体がさらに面白くなっていくことも期待できます。
