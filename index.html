ルール説明

まずはこの企画の基本ルールを確認してください。

  • プロ野球選手を株のように売買して資産を増やしていく企画です。
  • 選手の活躍によって価値が変動し、保有選手や売買タイミングで順位が変わります。
  • どの選手を買うか、いつ売るかという判断が非常に重要です。
  • 最終的に資産の大きい参加者ほど上位入賞に近づきます。

ルールをもっとわかりやすく知りたい方へ

詳しいルール解説はYouTubeでわかりやすく説明しています。 初参加の方は、まず動画を見てから参加すると理解しやすくおすすめです。

EVENT 01

初回参入ボーナス

新規参加者向けの初回ボーナスです。 初めて参加する方には、参入時に 50pt を付与します。 参加しやすくするためのスタート特典で、最初の売買を始めやすくする目的があります。

EVENT 02

開場記念ボーナス

開場記念として参加者にボーナスを付与するイベントです。 開始直後の参加ハードルを下げ、より多くの方が楽しめるように設定しています。 企画スタートの盛り上がりを後押しする記念特典です。

EVENT 03

ひな祭り5%還元キャンペーン

ひな祭り記念の期間限定イベントです。 対象期間中の取引では、5%還元の特典があります。 通常よりもお得に参加できる特別キャンペーンで、短期的に動く面白さも高まります。

EVENT 04

YouTube登録ボーナス

みるスタチャンネルを登録してくれた方向けのボーナスです。 チャンネル登録者が増えるほど企画全体も盛り上がり、今後の賞金アップにもつながっていく可能性があります。 企画を応援しながら、自分にもメリットがあるイベントです。

EVENT 05

Xリツイートボーナス

Xで対象投稿をリツイートしてくれた方向けのボーナスです。 企画の拡散に協力してくれた方へ還元するイベントです。 参加者が増えることで企画全体がさらに面白くなっていくことも期待できます。
